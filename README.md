EcoSort is an interactive AI chatbot application explicitly aligned with **United Nations Sustainable Development Goal 12: Responsible Consumption and Production**. It eliminates recycling confusion at the household level by instantly identifying the proper bin location for common items.

## 🚀 Project Overview & Architecture
The workflow functions using an event-driven loop structure built using the **Botpress Cloud** NLU framework and an autonomous vector knowledge base engine. 

- **Frontend/Interface:** Visual custom web chat element.
- **AI Knowledge Engine:** Botpress Autonomous Node querying a verified regional text vector index.
- **Open-Source Alternative:** Pure Python script fallback simulating keyword array vector mapping.

## 📂 Repository Contents
- `ecosort_bot.py`: Main executable backend script simulating the interactive knowledge matching engine.
- `README.md`: System documentation and architectural definitions.
